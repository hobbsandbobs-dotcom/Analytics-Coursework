# Analytics-Coursework

This investigation used Python, SQL within R, R analytics, and MongoDB Atlas to investigate into the operational inefficiencies and failures for the casestudy of NorthStar Logistics

The structure of the dataset repository is:
1_Data_importing_and_cleaning.ipynb  which is where the data was imported and cleaned  
2_Python_Analytics.ipynb which is where Python was used Python exploratory analysis and operational analytics  
3_SQL_and_R_analytics.ipynb   which is where SQL and R statistical analysis was used  
4_Mongo_DB_Section.ipynb   which is where the MongoDB Atlas was designed, and CRUD operations undertaken within it, as well as the aggregation pipeline and indexing 
raw    contains the original datasets provided to us  
cleaned   contains the cleaned datasets I prepared during the EDA phase, during 1_, and for use across the subsequent analytical stages
